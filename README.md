## [ ++ SYSTEM_OVERVIEW ++ ]

**KAKO_MAINFRAME** is not a standard portfolio. It is a highly immersive, interactive Web 1.0 nostalgia vault. Built completely from scratch using Vanilla HTML, CSS, and JavaScript, this project rejects modern, sterile web design in favor of raw, atmospheric, CRT-monitor hacker aesthetics. 

The site is segmented into highly secure "Vaults" or "Nodes," each housing custom media players, interactive lore typewriters, and gamified unlocking sequences. 

---

## [ ++ CORE_MODULES ++ ]

### 01. THE ENTRANCE PROTOCOL (`index.html`)
A strictly timed security checkpoint. 
* Fullscreen background GIF draped in heavy vignettes and CRT scanline overlays.
* Mechanical typewriter effect that types out the system greeting.
* Blood-red countdown timer. Once `[ 0 ]` is reached, a subtle screen glitch occurs, and the user is granted access to the main hub via the `[ ACCESS GRANTED ] >> INITIATE <<` command.

### 02. THE MASTER HUB (`h0me.html`)
The central nervous system of the site. Features a terminal-style link tree that triggers hidden `iframe` popups over the screen without ever leaving the page. 

### 03. THE LEGENDS VAULT (7-Star Summoning Engine)
A dedicated, gamified shrine to the GOATs: **Goku** and **Luffy**.
* Users must click through a circular layout of 6 glowing 3D Dragon Balls.
* Activating a node plays a specific fan-edit video or artwork.
* A neon `STAR CREATION LEVEL` progress bar tracks the user's journey.
* **The Grand Finale:** Unlocking all 6 balls summons the hidden 7th Dragon Ball. Clicking it triggers an earthquake animation and splits the screen down the middle, revealing a dual-typewriter sequence and hidden Ultra Instinct / Gear 5 special videos.

### 04. S0UNDTRACKZ PLAYER (The Music Vault)
A custom-built, retro TV music player.
* Features a custom playlist with interactive progress bars, volume sliders, and functional track navigation `[ << ] PLAY [ >> ]`.
* Each track dynamically changes the CRT screen's background GIF and mechanically types out the song name.

### 05. THE PHANTOM ARCHIVES (Chrollo's Requiem)
A lore-heavy interactive text vault dedicated to Chrollo Lucilfer.
* Left-hand navigation menu to cycle through iconic quotes.
* Right-hand display features a custom blood-red hacker scrollbar and a flawless mechanical typewriter that auto-scrolls through his dialogue and biography.

## [ ++ SYS.AUDIO_ENGINE_V3 ++ ]

This project features a highly complex, custom-built Background Music (BGM) Interceptor. 

* **Ambient Autoplay:** Streams directly from a Catbox host link the moment the hub is breached.
* **Smart UI:** A fixed terminal module `[ SYS.AUDIO // ACTIVE ]` allows the user to mute/unmute seamlessly.
* **Cross-Frame Override Failsafe:** Using `window.postMessage`, the background music actively listens to the hidden `iframe` vaults. If a user hits `PLAY` inside the S0undTrackZ player or opens a video in the Heroes Vault, the system physically locks the BGM button, flashes a yellow `OVERRIDE` warning, and silences the ambient track to prevent audio clashing. 
* **Auto-Resume:** Closing any vault via the `[ X ]` button instantly flushes the iframe memory, stops the hidden video/song, and fades the background ambiance back in.

## [ ++ INITIATION_SEQUENCE ++ ]

No installations, no frameworks, no local servers required.

1. Clone or download this repository to your local machine.
2. Locate `index.html` in the root directory.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Brave).
4. Wait for the security countdown to reach zero.
5. Click anywhere to initiate the mainframe.
